Portfolio Website

This is the website you are seeing now, which I made for a university assignment. As you can see, when the cursor is moved a pixel effect follows it. This effect is actually something called Conway's game of life. Each pixel represents a cell and they die or spread depending on their surrondings. Moving your cursor places cells below it. This was made in vanilla javascript with no framework.

B Tree Implementation in Rust

When first learning Rust I chose to implement a B tree as a project to learn the language. A B-tree is a type of data structure called a tree. A tree is a data structure which represents a heirarchy of nodes, with only one root node being at the very top. A B-Tree splits branches into ranges of indexes, which makes it very fast for searching across lots of nodes and a good choice for database systems.

Map of Spotify

In this project I created a kind of 'map' of spotify. I wrote a script that recursively builds a graph of spotify artists. The script would start at an artist A. It would then look at the artists that users who like A most often listen to. From there it would add A as a node then add these similar artists as nodes and link them to A. It would then go to the first similar artist B and add their similar artists as nodes and link them to B. It would then go to the second similar artist of A, C and do the same. Once it had linked all of A's similar artists it would repeat the process with B, then C and so on. Recursively building a graph of artists on spotify. I then visualized this data using a python graphing library.
